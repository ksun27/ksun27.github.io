Introduction

I am currently working as a Software Engineer II at FactSet on the Test Manager Infrastructure team. My job is to service our Test Manager platform, a Content Quality tool used to create, manage, submit and review data quality tests. It supports testing data integrity as well as data comparison of Templates (DOWNL), Screens (US), Queries (SQL), APIs (Postman Collections) and Jupyter Notebooks.

Role & Responsibilities

As a Software Engineer II, I work on maintaining and enhancing the Test Manager Infrastructure that enables data quality assurance across FactSet's financial data products. This involves developing tools and systems that help ensure the accuracy and reliability of financial data used by investment professionals worldwide.

Technologies & Skills

In this role, I work with various technologies and methodologies to build robust testing infrastructure, including data validation frameworks, API testing tools, and automated quality assurance systems that help maintain FactSet's high standards for financial data accuracy.
